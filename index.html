Calculadora

Instrucciones de la Calculadora

Esta es una calculadora simple que permite realizar operaciones básicas:

  • Sumar: Usa el botón "+" para sumar dos números.
  • Restar: Usa el botón "-" para restar dos números.
  • Multiplicar: Usa el botón "*" para multiplicar dos números.
  • Dividir: Usa el botón "/" para dividir dos números.
  • Borrar: Presiona "C" para limpiar la pantalla.
  • Resultado: Presiona "=" para calcular el resultado.
